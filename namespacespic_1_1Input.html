Enumeration Type Documentation

◆ KeyCode

enum spic::Input::KeyCode
strong

An enumeration describing the different keyboard keycodes.

These keycodes are taken from the usb.org HID specification. See: https://www.usb.org/sites/default/files/documents/hut1_12v2.pdf page 53

Incidentally, SDL also adheres to these values, which means you can cast the scancode from SDL to one of these values.

Attention
This is a symbol defined by the original SPIC API and should be kept as-is.
